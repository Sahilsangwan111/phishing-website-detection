# 🔐 Phishing Website Detection

## 📌 Overview

This project detects phishing websites using Machine Learning techniques based on URL features.

## ⚙️ Tech Stack

* Python
* Scikit-learn
* Pandas
* NumPy

## 🚀 Features

* Detects phishing and legitimate websites
* Fast prediction using trained ML model
* Easy to use interface

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python app.py
